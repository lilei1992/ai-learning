# AI Learning

个人 AI 学习资料库。

## 目录

- `ai-agent-runtime/`：AI Agent Runtime 每周目录版学习计划。
- `ai-agent-runtime-learning-plan.md`：AI Agent Runtime 总学习计划。
- `ai-agent-runtime-daily-goals.md`：56 天每日学习目标汇总版。
- `ai memory/`：AI memory 相关学习笔记。
- `claude code/`：Claude Code 相关学习笔记。
- `trade-system-engineering/`：交易系统工程能力学习计划，覆盖业务建模、系统链路和 AI 协作。

## 学习方式

- 按 `ai-agent-runtime/README.md` 的 Week / Day 顺序推进。
- 每天完成一个主题，忙时顺延，不按自然日期强行追赶。
- 每个 Day 文件底部保留“今日笔记”，用于记录复习要点、后端类比和待验证问题。
