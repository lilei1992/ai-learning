核心目标：

- 综合前 7 周内容，设计一个最小可行 Agent Runtime。

必须包含：

- Agentic Loop（智能体主循环）
- Context Manager（上下文管理）
- Tool Registry（工具注册表）
- Tool Executor（工具执行器）
- Permission Service（权限服务）
- Transcript Store（执行记录存储）
- Evaluation Harness（评估框架）
