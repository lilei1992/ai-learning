核心目标：

- 理解 Agent 需要独立评估体系。

关键概念：

- `Evaluation（评估）`：衡量 Agent 是否完成任务。
- `Benchmark（基准测试）`：一组固定任务，用于比较表现。
- `Golden Task（金标任务）`：预期结果明确的回归任务。
- `Success Criteria（成功标准）`：判断任务成功的规则。

需要理解：

- Agent 输出开放，不能只用简单断言。
- 评估要覆盖正确性、安全、成本、延迟、稳定性。
- 金标任务应长期保留，防止能力回退。

今日输出：

- 设计 5 个 Agent Golden Task。
